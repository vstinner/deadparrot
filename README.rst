++++++++++
deadparrot
++++++++++

Provide removed C API functions to new Python versions.

Provided functions:

* PyEval_CallObjectWithKeywords()
* PyEval_CallObject()
* PyUnicode_GetMax()

Support Python 2.7 and Python 3.6-3.13. PyPy is not supported.
